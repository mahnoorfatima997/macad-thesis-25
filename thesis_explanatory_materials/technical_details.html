Cognitive Metrics Framework

Six Core Cognitive Metrics

1. Cognitive Offloading Prevention (COP)

Definition: Measures the system's effectiveness in preventing users from offloading cognitive work to AI.

Calculation:
COP = (1 - direct_answers / total_interactions) × consistency_factor
where consistency_factor = 1 - std_dev(prevention_rates)

Target: > 70% (Based on UPenn research showing 48% baseline)

Importance: Prevents cognitive dependency and promotes independent thinking.

2. Deep Thinking Engagement (DTE)

Definition: Quantifies engagement in analytical and critical thinking activities.

Calculation:
DTE = Σ(thinking_interactions × complexity_weight) / total_interactions
complexity_weight = 1 + (question_count × 0.2) + (response_length / 100 × 0.1)

Target: > 60% (Belland et al. 2017 meta-analysis baseline: 42%)

Importance: Indicates quality of cognitive engagement.

3. Scaffolding Effectiveness (SE)

Definition: Measures adaptive support provision based on user skill level.

Calculation:
SE = Σ(scaffolding_quality × skill_appropriateness) / interactions
skill_appropriateness = 1 - |expected_level - actual_level| × 0.3

Target: > 75% for optimal learning support

Importance: Ensures appropriate challenge level (Zone of Proximal Development).

4. Knowledge Integration (KI)

Definition: Tracks synthesis of information from multiple sources.

Calculation:
KI = (unique_concepts × connection_strength) / total_concepts
connection_strength = avg(semantic_similarity) × temporal_coherence

Target: > 50% for effective learning

Importance: Indicates deep understanding and concept linking.

5. Learning Progression (LP)

Definition: Measures skill development over time.

Calculation:
LP = (skill_level_end - skill_level_start) / time_elapsed × progression_consistency
progression_consistency = 1 - variance(skill_measurements)

Target: Positive slope with r² > 0.6

Importance: Validates educational effectiveness.

6. Metacognitive Awareness (MA)

Definition: Evaluates self-reflection and learning strategy awareness.

Calculation:
MA = (reflection_count × depth_score) / interactions × strategy_diversity
depth_score = avg(reflection_length) × semantic_complexity

Target: > 40% (STEM baseline: 31%)

Importance: Critical for autonomous learning.

Anthropomorphism Prevention Metrics

Dependency Detection

Identifies patterns of over-reliance on AI assistance.

  • Approval-seeking ratio: Questions ending with validation requests
  • Decision delegation: Direct requests for AI to make choices
  • Emotional attachment indicators: Personal pronouns, gratitude expressions

Autonomy Promotion

Measures system's success in promoting independent decision-making.

  • Self-directed exploration rate
  • Independent problem-solving attempts
  • Reduction in confirmation-seeking over time

Language Pattern Analysis

NLP-based detection of anthropomorphic language.

  • Personal attribution ("you think", "you believe")
  • Emotional projection ("you feel", "you want")
  • Social relationship markers

Calculation Methodologies

Aggregate Score Calculation

Overall Effectiveness Score:
E = Σ(w_i × m_i) / Σ(w_i)
where:
w_i = weight for metric i (based on importance)
m_i = normalized metric value (0-1 scale)

Weights:
COP: 0.25, DTE: 0.20, SE: 0.20, KI: 0.15, LP: 0.10, MA: 0.10

Improvement Over Baseline

Improvement Calculation:
I = ((measured - baseline) / baseline) × 100

Statistical Significance:
t = (measured - baseline) / (std_error / √n)
p < 0.05 for significance

Temporal Analysis

Learning Curve Fitting:
Performance(t) = a × (1 - e^(-b×t)) + c
where:
a = asymptotic performance level
b = learning rate
c = initial performance
t = time or interaction count

Graph Machine Learning Pipeline

Graph Neural Network Architecture

Node Feature Encoding (16 dimensions)

Dimension Feature Encoding Method
0-2 Interaction Type One-hot encoding (question/response/reflection)
3-5 Agent Type One-hot encoding (5 agents)
6 Cognitive Load Normalized float (0-1)
7 Complexity Score Normalized float (0-1)
8-10 Cognitive Metrics COP, DTE, SE values
11 Temporal Position Normalized timestamp
12-15 Content Embedding TF-IDF top components

GNN Layer Architecture

Multi-Layer Message Passing:
Layer 1: GAT (Graph Attention) - 16 → 32 features
Attention: α_ij = softmax(LeakyReLU(a^T[Wh_i || Wh_j]))

Layer 2: GraphSAGE - 32 → 16 features
Aggregation: h_i' = σ(W · [h_i || AGG({h_j : j ∈ N(i)})])

Layer 3: GCN (Graph Convolution) - 16 → 8 features
Convolution: H' = σ(D^(-1/2)AD^(-1/2)HW)

Clustering Algorithm

Proficiency Clustering

K-means clustering on graph embeddings with k=4 (Beginner, Intermediate, Advanced, Expert)

Silhouette Score Optimization:
s(i) = (b(i) - a(i)) / max(a(i), b(i))
where:
a(i) = average distance to points in same cluster
b(i) = minimum average distance to points in other clusters

Link Strength Calculation

Semantic Similarity:
strength = cosine_similarity(embed(move_i), embed(move_j)) × temporal_weight
temporal_weight = exp(-λ × |t_i - t_j|)
λ = 0.1 (decay parameter)
