Core Metric Formulas (Click to expand)

Cognitive Offloading Prevention (COP): COP = (prevented_direct_queries / total_queries) × inquiry_depth_weight

Deep Thinking Engagement (DTE): DTE = Σ(response_complexity × time_spent × reflection_indicators) / total_interactions

Scaffolding Effectiveness (SE): SE = Σ(guidance_appropriateness × user_progress) / total_scaffolding_events

Knowledge Integration (KI): KI = (integrated_sources + concept_connections) / total_knowledge_opportunities

Engagement Consistency (EC): EC = 1 - (std_dev(response_times) / mean(response_times))

Anthropomorphism Metrics (Click to expand)

Cognitive Autonomy Index (CAI): CAI = autonomy_ratio - 0.5 × dependency_ratio

Anthropomorphic Dependency Score (ADS): ADS = (personal_attribution_count / total_words) + emotion_language_score

Professional Boundary Index (PBI): PBI = 1 - (off_topic_ratio + personal_conversation_ratio)

Neural Engagement Score (NES): NES = (unique_concepts_count / total_interactions) + technical_vocabulary_ratio
